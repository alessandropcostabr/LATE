# Manual de Uso — tmux (Administração de Cluster)

**Data:** 09/11/2025

## Por que usar
- Executar comandos simultaneamente (mach1/2/3)
- Sessões persistentes
- Monitorar múltiplos serviços

## Início
- `cluster` → roda `~/cluster_tmux.sh` com 3 painéis e sincronização

## Comandos
- Prefixo: `Ctrl+b` + (tecla)
- Divisão de painéis, alternância, redimensionamento, sync on/off

## Troubleshooting
- Sessão duplicada, problemas de SSH, cores, script ausente
